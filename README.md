#使用方法 
###1 构建本机apache服务器 并在根目录创建Preview 文件夹 
#### 建议 对将要存储的文件创建 file 文件夹存储
###2 使用fileEnumator获取文件列表
###3 获取服务器地址 如手机与服务器在同一局域网内建议使用服务器的DHCP地址
